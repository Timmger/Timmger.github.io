1. 概述

动态规划主要应用于纯递归问题的优化。只要遇到对相同输入的重复调用时，我们都可以使用动态规划来优化它，主要的思路就是简化子问题结果的存储,这样当之后求解子问题时，我们就不用重复计算。
动态规划算法可以将时间复杂度从指数时间降到多项式时间，通常有以下4个步骤来设计：

  1. 刻画一个最优解的结构特征
  2. 递归地定义最优解的值
  3. 计算最优解的值，通常采用自底向上的方法
  4. 利用计算出的信息构造一个最优解

2. 最长递增子序列

问题描述：
在最长子序列中，输入是一串数列 $a_1$, … , $a_n$。子序列是这些数的任意子集，形如$a_{i_1},a_{i_2},{…} ,a_{i_k}$ ,其中 $1\le{i_1}\lt{i_2}\lt{…}\lt{i_k}\le{n}$，递增子序列是数字严格增大的子序列。我们需要找到最长的递增子序列。例如：

{6,1,8,5,3,6,9,7}的最长递增子序列为{1,3,6,9}

问题分析：
为了更好地理解问题的解空间，我们构造一个有向无环图(dag)：
为每个元素$a_i$构造一个节点$i$, 并建立一条边｛$i$,$j$｝，且${a_i}\le{a_j}$, ${i}\lt{j}$
