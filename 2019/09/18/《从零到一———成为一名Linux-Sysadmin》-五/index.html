Bash脚本编程入门

1. 一个bash脚本

helloworld.sh

添加others执行权限

chmod +x ./helloworld.sh

运行脚本

bash ./helloworld.sh 或 ./helloworld.sh

执行并封装公用代码，可导入另一个脚本的变量

source ./helloworld.sh 或 . helloworld.sh

定义变量：

变量名与等号之间不能有空格, 命名只能使用英文字母，数字和下划线

使用变量时，在变量名前加入美元符号 $, 可使用括号限定边界

使用 unset 可以删除变量

$1..$n : 输入传递的第n个参数

$@ : 从输入传递的所有参数
