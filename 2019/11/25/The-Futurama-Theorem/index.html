The Futurama Theorem

动画里的数学

今天在看《飞出个未来》(S06E10. The prisoner of Benda)，编剧大大 Ken Keeler 把一个很有意思的数学问题带到了故事情节里面。
Professor 发明一个心智转换仪，能够交换两个个体的心智, 经过一番讨论，他和实习生 Amy 决定交换他们的大脑，但事后，当他们要换回来的时候，机器却失效了，博士解释为由于大脑的免疫反应，一旦两个人交换心智后，他们就永远不能交换回来。
但博士提出一个假设：或许可以通过第三方个体交换回来，此时，Bender 过来自愿当 “temporary storage”, 它和 Amy（心智为博士） 交换了大脑，然而博士却发现，这个方法不行，如果 Bender(此时为博士心智) 和 博士(Amy心智) 交换的话，博士虽然还原了，但是 Bender 和 Amy 不能再交换了，因为他们已经交换了一次。
随着故事的发展，越来越多的人进行了交换，最后形成这样一个交换链：

那么问题来了，怎么才能把所有人还原呢？故事的最后，编剧给出了答案：

我们只需要引入两个 “new bodies” {x, y}, 就可以把所有人都还原(没给出证明过程)。

首先，我们要在所有人当中找到这样一个交换链：第二个人心智在第一个人身体里，第三个人心智在第二个人身体里，… ，第 k 个人心智在第 k-1 个人身体里，而第一个人心智在第 k 个人身体里，他们构成这样一个 K 环，(相当于把原来所有人心智前移一位，第一个人心智移到了最后)。OK, 我们可以在所有人当中找到两个这样的环。

之后，引入两个temp元素 {x, y}, 对任意的 $i = 1, 2, .., k$ 存在这样一个交换序列 $\sigma$, 可以还原所有元素（环内部所有的元素都不可以直接交换，只能与外部元素 {x, y} 交换一次）。

$$\sigma = (<x, 1> <x, 2> … <x, i>)(<y, i+1> <y, i+2> … <y, k>)(<x, i+1>)(<y, 1>)$$

其中, <a, b> 代表交换索引a 和 索引b 的内容。

OK, 对于编剧给出的公式，我们来实验一下，对于第一个环（7-cycle）, 不妨取 $i = 1$ :

$$\sigma = (<x, 1>)(<y, 2> <y, 3> … <y, 7>)(<x, 2>)(<y, 1>)$$


由图示所见，根据序列交换后，除了 x, y 没有还原，其他都还原了，而 x, y 本身没有交换过，可以再交换一次还原，所以只有一个环时，总交换次数为 $k + 3$。而存在两个环时 x 和 y 不需要再交换，总次数为 $(k_1 + 2) + (k_2 + 2)$。
我们推广到奇数个环 n-cycle，n取奇数，总次数为 ：

$$(k_1 + k_2 + … + k_{n-1}) + 2(n-1) + k_n + 3$$

n-cycle 为偶数个环, n-cycle 总次数为：

$$(k_1 + k_2 + … + k_n) + 2n$$

由此可知，在本片中，存在两个环 7-cycle 和 2-cycle，总交换次数为 13。
